# 📘 Assignment: Python Classes

## 🎯 Objective

Learn how to define and use classes in Python to model real-world objects and behaviors. Students will practice creating attributes, defining methods, and working with class instances.

## 📝 Tasks

### 🛠️ Create a Basic Class

#### Description
Create a `Car` class that stores information about a car and can display its details.

#### Requirements
Completed program should:

- Define a class named `Car`
- Include `make`, `model`, and `year` attributes
- Add a `display_info()` method that prints the car's details
- Create an instance of `Car` and call `display_info()`

### 🛠️ Add Instance Methods

#### Description
Extend the `Car` class with mileage support so the object can track and display updated mileage.

#### Requirements
Completed program should:

- Add a `mileage` attribute to the `Car` class with a default value of `0`
- Add an `update_mileage(new_mileage)` method to update the mileage
- Add a `display_mileage()` method to print the current mileage
- Demonstrate updating and displaying mileage for a `Car` instance
